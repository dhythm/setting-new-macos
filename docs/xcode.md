# Xcode

Apple の統合開発環境（IDE）。

## ダウンロード

https://apps.apple.com/us/app/xcode/id497799835?mt=12

## インストールの推奨

Xcode は各種環境構築で必要になることが多いため、早めにダウンロードしておくことを推奨します。

### 開発コマンドツール（Command Line Tools）

Xcode をインストールすると、開発コマンドツール（Command Line Tools）も同時にインストールされます。
多くの開発ツールやライブラリのビルドで必要なコマンドラインツールが含まれています。以下のようなツールを使用する際に必要です。

- Homebrew
- Git
- 各種コンパイラ（gcc, clang など）
- その他の開発ツール

iOS や macOS 向けの開発をしない場合でも、環境構築において Command Line Tools が必要となることが多いため、Xcode のインストールを推奨します。
