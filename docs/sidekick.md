# Sidekick

Chromium ベースのブラウザ。
サイドバー（無料版では 5 つまで）と設定の同期が便利。

2025/08/03 でサポート終了。
[Comet](./comet.md) に移行が推奨される。

## 設定

Google アカウントでログインすると設定を共有可能。
`Settings > Profile > Sync and services` にて設定を同期させる。

### `Sync Settings`

Customize sync を選択。

- Bookmarks
- Extensions
- History
- Settings
- Theme
- Open tabs
- Passwords and passkeys
